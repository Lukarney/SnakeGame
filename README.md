# SnakeGame
A fun snake side scrolling game
that implemets sidescrolling, drag and drop, and classes.
The goal of the game is to get to the snake's hole. You can collect hearts to increase your lives. Luckily, you can drag and drop it near you.
If you run into a cactus you will lose a life, unless it's a friendly cacti.The cactus will dissapear after touching it. If you loose all of your lives you will die and loose. Afterwards, you can press r to restart 
press 'h' for help and shift-s 'S' for help in your terminal.
